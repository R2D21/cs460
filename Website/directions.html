CatfishC Compiler

  • Directions

    • Compile (From PA5)

      • >$ cd build

      • >$ make

    • Run (From Build)

      • >$ cd ../bin

      • >$ ./catfishC -c test.c

        • Optional flags:
          • --help Print usage and exit.
          • --assembly, -S Output assembly.
          • --3AC, -q Output 3 address code.
          • --compile [file], -c [file] Compile [file].
          • --output [file], -o [file] Output to [file].
          • -d[l|y|s] Output lex/yacc/symbol table.
        • Example:
          • $ ./catfishC -c test.c -dly -q
