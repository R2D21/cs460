  • 3AC Info

    • Naming convention of 3AC variables

      • Local Variable at byte offset #
        • LOC_V_#
      • Global Variables at byte offset #
        • GL_V_#
      • Integer Ticket Counter for temporary register
        • IT_#

    • Note:

      • We followed the recommended 3AC on the Intermediate Language Specification document
      • We did not implement functions or pointers
      • We did implement for/do while loops
      • We only generate 3AC for integer data types
