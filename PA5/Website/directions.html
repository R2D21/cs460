CatfishC Compiler

  • Directions

    • Compile (From PA5)

      • >$ cd build

      • >$ make

    • Run (From Build)

      • >$ cd ../bin

      • >$ ./krwCompiler testfile.c -flags

        • Optional flags:
          • -dyl (optional y to output reductions, optional l to output tokens)
          • -o output (to specify output file name)

            • Note: Custom output file is not updated on website. To see updated files, omit -o flag.

        • Examples:
          • $ ./krwCompiler testfile.c -dy
          • $ ./krwCompiler testfile.c -o myoutput -dly

    • Symbol Table Test Driver (From PA5)

      • >$ cd SymbolTableTest

      • >$ make

      • >$ ./TestST
